# Day 8 â€“ Iterators and Generators

## What I Learned Today
- How iterators work using iter() and next()
- How to create custom iterators
- How generators use yield
- What generator expressions are

## Key Concepts
- Iterator gives values one at a time and remembers position
- iter() turns an iterable into an iterator
- next() gets the next value from an iterator
- Custom iterators must define __iter__() and __next__()
- StopIteration ends iteration
- Generators use yield instead of return
- Generator expressions create values lazily using ()

## Why This Matters
- Saves memory
- Handles large data efficiently
- Makes iteration cleaner and easier
